Mission Control

Opinionated JavaScript framework for building real-time dashboards

Focus on

building widgets and connecting your APIs and services

and let Mission Control handle

Real-time Sync User Management Role-based permissions HTTP & WebSocket transports And more...

Beginning a real-time web project, like building your own home dashboard, can be a tedious and time-consuming process.

Coding lots of boilerplate for authentication, authorization and data communication isn’t fun at all. By the time you get to building the thing you wanted you got bored and moved on to the next.

Mission Control aims to provide all those features out-of-the-box so you can get to creating what matters right away.

Quick Start

Install Mission Control from NPM.

$ npm i -g @capevace/mission-control

Start it up as a standalone server

$ mission-control start

Import it as a JS framework

const { init } = require('@capevace/mission-control');
const missionControl = await init({
    /* Options... */
});

Easy to use Plugin API

Extend Mission Control and connect your APIs and services

Mission Control provides an intuitive and easy to use plugin API that you can use to build your own features or connect to external services to fetch data.

Common features provided out-of-the-box

Build that thing you want and let Mission Control handle the rest

Mission Control has common features already built-in so you don’t have do. Get straight to building those things that are important for your dashboard, like monitoring Dogecoin prices ($0.376, btw).

React-like state updates

Real-time state synchronization

Create data services that automatically synchronize data changes to connected clients (opened dashboards).

Bandwidth usage is kept low by only transmitting the data that actually changed. This makes using dashboards on-the-go a breeze!

Data endpoints are automatically available via HTTP and WebSocket, with the latter supporting push updates.

Authentication & Authorization

User management &
role-based permissions

Don’t worry about managing user passwords or admin accounts. Secure your dashboard with built-in permissions & user management.

Database & Validation API

Validate with Joi and persist data using SQLite

Data can be saved and preserved between launches by using the Database API. SQLite is used to store data permanently, including files.

Mission Control also integrates with Joi to properly validate and sanitize input data.

Command-Line Interface

Intuitive CLI for use as standalone server

The recommended way to use Mission Control is to use it as a standalone server and extend it using its Plugin API.

However it can also be imported like any JS framework for more heavily customized projects.
