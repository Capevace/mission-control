Module: @socket

The Socket Module

The socket module handles most communications with clients. Pure HTTP is actually used very rarely within Mission Control, even though the interfaces for it are actually there.
Sockets have the advantage that they're real-time, which is very useful for home automation. It also acts as a sort of networked message bus by broadcasting events.
